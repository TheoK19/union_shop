# Union Shop

A Flutter-based mobile application for the university union shop.

## Features

- View a list of product collections.
- User authentication (login and sign up).
- A simple and intuitive user interface.

## Getting Started

### Prerequisites

- Flutter SDK: https://flutter.dev/docs/get-started/install
- A code editor like Visual Studio Code or Android Studio.

### Installation

1.  Clone this repository:

    ```bash
    git clone https://github.com/your-username/union-shop.git
    ```

2.  Navigate to the project directory:

    ```bash
    cd union-shop
    ```

3.  Install the dependencies:

    ```bash
    flutter pub get
    ```

### Running the Application

1.  Open the project in your code editor.
2.  Run the application on an emulator or a physical device:

    ```bash
    flutter run
    ```
